# AI-Based Data Analytics Platform

An AI-powered data analytics platform built using Python and Streamlit for visualization, prediction, and intelligent insights from CSV datasets.

# Features
- Data Visualization
- Prediction Module
- Technical Analysis
- AI Insights
- PDF Report Generation

# Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
